# Gallery

A learning project for React development.

## Technologies

- **JavaScript** (90.9%)
- **HTML** (5.1%)
- **CSS** (4%)

## Getting Started

This project is built with React and Vite for fast development and hot module replacement.

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

## Purpose

This repository is a learning project to practice React fundamentals and build interactive components.
